# Banijjik Project Agent Rules

This directory contains the machine-readable and human-readable rules, instructions, and workflows for the **Banijjik** SaaS platform.

## Structure

- `instructions.md`: Core architectural rules, naming conventions, and project philosophy.
- `workflows/`: Step-by-step guides for common tasks (e.g., adding domains, fixing bugs).

## How to use

As an AI agent, you must read `instructions.md` before performing any architectural changes. Follow the workflows in the `workflows/` directory for consistency.
